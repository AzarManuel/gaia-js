ABOUT

Gaia is a site modelling system which allows the user to read elevation data for any site, directly from Google Maps, and then processes this information to build a 3D terrain model which illustrates dynamic patterns of environmental factors.

The elevation data, acquired through the Endeavour Shuttle SRTM (Radar Terrain Mapping) missions provided by NASA.

DISCLAIMER

This system is a coarse first approximation towards simulating environmental processes. It was made for educational purposes.

It may not be used for legal, planning or commercial matters.
